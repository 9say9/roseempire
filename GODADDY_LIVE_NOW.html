Make roseempire.co.uk live — 3 DNS edits

Your site is already on Netlify. GoDaddy DNS must point to Netlify. Email stays working if you only edit the rows below.

Step 2 (turn off Launching Soon): You do NOT need a separate page for this. Changing DNS in Step 3 removes the GoDaddy placeholder automatically. Skip Step 2 — go straight to DNS below.

Open GoDaddy DNS for roseempire.co.uk

Edit 1 — A record (Name = @)

  • Find row: Type A, Name @
  • Click the pencil (Edit)
  • Change Value to:
75.2.60.5
  • Save
  • If there is a second A record on @ (e.g. 13.248.243.5 or 76.223.105.230) — click Delete on that extra row

Edit 2 — www record

  • Find row: Name www
  • If Type is A or points to GoDaddy IP — click Edit, change Type to CNAME (or delete A and Add new CNAME)
  • Set Value to:
say9.netlify.app
  • No https:// — just the text above
  • Save

Do NOT touch

  • Keep all MX rows (email)
  • Keep all TXT rows (email)

After saving — Netlify

Open Netlify Domain settings

  • Add domain roseempire.co.uk if not added
  • Add alias www.roseempire.co.uk
  • Wait 30–90 minutes → status Verified + HTTPS
  • Set primary: www.roseempire.co.uk

Test when ready

  • www.roseempire.co.uk → Rose Empire site
  • If still "Launching Soon" → wait longer or send DNS screenshot to Cursor
