Drop your UK test report certificate here (PDF or image scan).

Also supported:
  D:\roseempire\certificates\
  D:\roseempire certificates\   (your existing folder on D: drive)

Then double-click deploy_certificate.bat in the main roseempire folder,
or tell Cursor: "certificate is in the folder, deploy it".

The file will be published as:
  assets/Rose-Empire-BS-EN-12935-Pillow-Report.pdf
  https://www.roseempire.co.uk/assets/Rose-Empire-BS-EN-12935-Pillow-Report.pdf

Tip: You can also drag a PDF into Cursor chat, but the easiest way is to
copy it into one of the folders above and run deploy_certificate.bat.
